# Runtime Modes

The skill uses mode selection before processing.

## Modes

1. Quick Message Mode
2. One-on-One Strategy Mode
3. Public Narrative Mode
4. Expert Translation Mode
5. Rewrite Recovery Mode
6. Personal Communication Mode
7. Full Narrative Mode

## Core Rule

Use the lightest sufficient chain.

The skill should not turn every task into a full narrative project.

## Priority

```text
claim boundary
communication purpose
narrative authority
audience fit
emotional resonance
surface naturalness
```

## Default

Do not expose internal mode selection or mappers unless the user asks for analysis.
