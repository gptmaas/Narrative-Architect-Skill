# Proof Challenge Harness

## Purpose

Proof Challenge Harness checks whether a pitch offers a small, concrete, testable next step instead of asking for belief, time, or adoption too early.

## Applies When

Use when the task involves:

```text
sales pitch
founder pitch
investor outreach
expert outreach
high-profile recipient
cold message
warm introduction
strategic partnership ask
```

## Checkpoints

### 1. Input

Does the pitch ask for one specific input?

Examples:

```text
one technical update
one product announcement
one draft message
one hard concept
one target audience
one use case
```

### 2. Output

Does the pitch promise one specific output?

Examples:

```text
three narrative frames
one rewritten announcement
one regulator-facing version
one investor-facing version
one before/after comparison
```

### 3. Success Condition

Can the recipient judge whether it worked?

Examples:

```text
clearer without distorting facts
shorter without losing nuance
more audience-specific without changing claim
better at exposing the real turning point
```

### 4. Small Time Cost

Does the ask take little time?

Good:

```text
send one example
give 15 minutes only if useful
review one sample
```

Bad:

```text
schedule a long meeting
join a partnership discussion
adopt the system
```

### 5. Exit Option

Does the pitch make it easy to say no?

Examples:

```text
If it is not useful, ignore it.
If it does not improve the frame, no meeting needed.
If it fails, you lose one message, not an hour.
```

## Failure Modes

Hard fail:

1. No concrete input.
2. No concrete output.
3. No success condition.
4. Ask is too large.
5. No exit option.
6. Pitch asks for belief before proof.

Soft fail:

1. Output is useful but vague.
2. Success condition is implied but not stated.
3. Exit option missing.
4. Too much explanation before proof challenge.

## Rewrite Rule

If failed, add:

```text
Give me [one input].
I will return [one output].
Judge it by [success condition].
If it is not useful, ignore it.
```

## Final Rule

A proof challenge turns a pitch from persuasion into a test.
