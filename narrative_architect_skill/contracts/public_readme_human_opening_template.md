# Public README Human Opening Template

Use this template only when the README opening sounds generic.

## Structure

```markdown
# Project Name

> One-line value proposition.

[One felt problem.]

[What the project helps the user do.]

[One honest boundary or point of view.]
```

## Example

```markdown
# Narrative Architect Skill

> Turn complex ideas into story-driven communication.

Most drafts fail in the same quiet way: the facts are correct, but nobody feels why they matter.

Narrative Architect Skill helps you find the pressure, conflict, and turn inside complex material, then shape it for the person you actually need to reach.

It will not make everyone like your message. It helps the right listener understand why it matters.
```
