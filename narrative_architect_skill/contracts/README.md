# Robert Skill Output Contracts

Robert Skill uses output contracts when the user requests reusable artifacts.

## Current Contracts

```text
short_explanation_output_contract.md
github_readme_output_contract.md
```

## Contract Selection

Short explanation:

```text
Use when user asks for a brief explanation, especially for non-technical audiences.
```

GitHub README:

```text
Use when user asks for README, GitHub repo docs, open-source project description, or developer-facing project page.
```

## General Rule

Artifact contracts override style preferences when format, truthfulness, or usability is at risk.
